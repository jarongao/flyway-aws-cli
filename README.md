# Description

- This docker image is a [Flyway](https://hub.docker.com/r/flyway/flyway) image with [AWS CLI V2](https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-welcome.html) installed.
- This is used as a continuous delivery solution for [AWS RDS](https://aws.amazon.com/rds/) migrations.
- [GitHub link](https://github.com/jarongao/flyway-aws-cli)
- [Docker Hub link](https://hub.docker.com/r/jarongao/flyway-aws-cli)
